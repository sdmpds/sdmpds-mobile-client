## Getting Started

Clone Repo

````
https://github.com/sdmms/sdmms-mobile-client.git
````

Install dependecies for react native

````
cd sdmms-mobile-client
npm install
````

````

````

## Android

````
navigate to sdmms-mobile-client
react-native run-android
````
